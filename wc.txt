Өөрсдийн бүтээж буй вебийнхээ нэг хуудсыг сонгож аваад  (хэт жижиг биш) веб компонент (wc) хэлбэрт шилжүүлнэ. 
Тухайн хуудас нь өгөгдлийг жагсаасан, өгөгдөл нь like эсвэл сагслах гэх мэт үйлдэлтэй байх хэрэгтэй. 
Жишээлбэл, барааны жагсаалт, бараа нь сагслах боломжтой, сагсанд хийхэд сагсны утга шинэчлэгддэг байх.


1. App модуль, өгөгдлийг сүлжээнээс авчирдаг, илгээдэг модультай байна. (js2 shg neg js-tei bn )
2. Дор хаяж хоорондоо хамааралтай 2 хэсэг байх хэрэгтэй. Жишээ нь барааны ангилалуудыг wc болгож хийгээд сонгогдсон үед нь барааны жагсаалт шинэчлэгдэж байх. (2 оноо)
3. Компонентууд нь дор хаяж 2 аттрибуттай байх, аттрибут солигдоход өөрчлөлт шууд wc доо хувиралт өгч байх. (1 оноо)
4. Компонентууд нь Property тай байх. Аттрибут солигдоход ордог өөрчлөлтийг эсвэл өөр ямар нэг шинж чанарыг Property ашиглан өөрчилдөг байх. (1 оноо) 
5. Аль нэг компонентд CustomEvent үүсгэх бөгөөд өөр нэг компонентд тэр event тэй холбоотой өөрчлөлт ордог болгоно. Сагсанд орсон барааны мэдээллээс НИЙТ үнэ гаргаж байгаа компонент нь боддог ч юмуу (1 оноо)
6. Компонент нь үндсэн dark болон light mode ыг дэмждэг бөгөөд dark mode үед background-color болон текстийн өнгийг үндсэн вебийнхээ CSS ийн custom property оор дамжуулж авдаг байх. (ok) 
7. Компонентдоо дор хаяж 1 ShadowDom ашигласан байх. (ok)
8. !!! 1 Template, 2 slot ашигласан байх-> biy daaasan uur bh(style n uur ch ymuu) ymaa template slotoor hiivel zuer ym shg bn.

9. Компонент нь төлөвөө хадгалдаг байх. Browser-ийг хаагаад нээхэд сагсан дахь бараа байж байх жишээтэй. !
WEBAPI

Урьд нь ашиглаж байсан jsonbin.io зэрэг API ашиглаж хийсэн API аа өөрсдөө Node.js ашиглан ашиглан REST WebAPI аар хандах боломжтой болгоно.

Хийх ажлууд:
PostgreSQL санд өгөгдлийн сангаа байгуулна.
Node.js + expressjs ашиглан API endpoint уудаа үүсгэнэ
Өгөгдлийг жагсаалтаар авах (барааны жагсаалт, захиалгын жагсаалт, гм)
Өгөгдөл нэмэх (post, бараа, коммент, user нэмэх гм)
Өгөгдөл хайх, шүүх (Login хийх, бараа хайх, барааг шүүх гм.)
Өгөгдөл, Route тэй ажиллах js module ууд байдлаар кодоо бичнэ
Swagger UI ашиглан API аа documentation тэй болгоно.
